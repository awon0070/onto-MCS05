Introduction to Ontology of Heart Disease

Ontologies are a type of data structure that sorts objects into a hierarchy, as well as grouping similar objects together. This characteristic of ontologies is particularly useful in fields such as biomedicine, where there are huge amounts of data, such as genetics, cell types, diseases, and many more. For heart diseases in particular, there are many diseases that share common symptoms, as well as diseases with many similarities with each other. An ontology can help categorize and standardize all the diseases and help in diagnosing heart diseases. This project aims to enable non-ontology experts to be able to take advantage of ontologies by designing a web application that allows users to interact with an ontology.
